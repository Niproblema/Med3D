Med 3D is a free open-source 3D medical data visualisation application with support for displaying 3D mesh models (in form of OBJ files) as well as 3D volumetric data - such as CT and MRI scans (in form of MHR files).

Application also offers 3D stereoscopic display of data suitable for displays with option of interlaced stereo view.

User can interact with application with use of 3D mouse (3D Connexion) or with Leap Motion controller.

About

The project is being developed at Laboratory for Computer Graphics and Multimedia, Faculty of Computer and Information Science, University of Ljubljana. Project is developed mostly as work of students as part of their seminar work, thesis work and summer internship work.

The project is lead and coordinated by Ciril Bohak.

Development history:

The original application was developed by Simon Žagar (Thesis - in Slovenian) in 2012 as part of his BSc Thesis work at University of Ljubljana, Faculty of Computer and Information Science. He developed 3D visualisation environment and navigation, loading OBJ files and interlaced stereoscopic display.

Afterwards the application was extended and adapted by several authors as stated below:

  • Matevž Ropret - contributed integration of several separately developed methods into the application. He also implemented new design into the application.
  • Matjaž Krmelj, Pia Jerkič, Ciril Bohak - put together design of the application and website.
  • Žiga Lesar - contributed implementation of raycasting method for direct visualisation of volumetric data. (Thesis - in Slovenian)
  • Boštjan Čotar - contributed implementation of subdivision of volumetric data approximation for better conversion to 3D mesh model. (Thesis - in Slovenian)
  • Miloš Lukić - contributed implementation of Multiple Partitions of Unity Implicits (MPUI) algorithm - an alternative to MC - for transforming volumetric data into 3D mesh model. (Thesis - in Slovenian)
  • Peter Škrlj - contributed integration of 3D mouse support (3D Connexion) for navigation, integration of Leap Motion controller. He also conducted an usability study on using different input devices for interaction. (Thesis - in Slovenian)
  • Anže Sodja - contributed code refactoring, implementation of MHD reader for reading raw volumetric data, implementation of Marching Cube (MC) algorithm in Java and integration of MC algorithm in OpenCL for transformation of volumetric data into 3D mesh model. (Thesis - in Slovenian)
